# 🔧 GarageSys - Professional Garage Management

A full-stack vehicle service tracking system built with Django and Bootstrap 5.

### ✨ Features
- **Role-Based Access:** Dedicated portals for Admin, Mechanic, and Customers.
- **Service Journey:** Real-time horizontal timeline for repair progress.
- **Digital Documentation:** Mechanics can upload repair images and technical notes.
- **Secure Payments:** Integrated Razorpay for seamless invoicing and billing.
- **Analytics:** Revenue reporting and service trend charts for Admins.

### 🚀 Tech Stack
- **Backend:** Python / Django
- **Frontend:** HTML5, CSS3, Bootstrap 5, Chart.js
- **Database:** SQLite (Development)
- **Payment Gateway:** Razorpay
